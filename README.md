# MyFirstApp - Android Internship Task 1

## Overview

This project was created as part of Task 1 of the Android Development Internship program.

The objective of this task was to set up the Android development environment, learn Android Studio basics, and create the first Android application using Kotlin.

## Tools Used

* Android Studio
* Kotlin
* Android SDK
* GitHub

## Task Completed

* Installed and configured Android Studio
* Set up Android SDK and required components
* Enabled USB Debugging on Android device
* Connected a physical Android device to Android Studio
* Created a new Android project named **MyFirstApp**
* Successfully built and ran the application on a physical Android device

## Project Structure

* MainActivity.kt
* AndroidManifest.xml
* Resource files
* Gradle build files

## Learning Outcomes

Through this task, I learned:

* Android Studio setup and configuration
* Android project structure
* Kotlin basics for Android development
* Running applications on a physical Android device
* Basic GitHub repository management

# MyFirstApp - Android Internship Task 2

## Overview

This project was created as part of **Task 2** of the Android Development Internship program.

The objective of this task was to design and implement a modern **Login Screen UI** using **Jetpack Compose** in Android Studio, following Material 3 design principles.

---

## Tools Used

* Android Studio
* Kotlin
* Jetpack Compose
* Material 3
* Android SDK
* GitHub

---

## Task Completed

* Designed a responsive Login Screen UI
* Added Email input field
* Added Password input field
* Implemented Password Visibility Toggle
* Added "Forgot Password?" option
* Created a Login button with loading indicator
* Implemented Snackbar messages for login success and failure
* Applied Material 3 design components
* Tested the application on an Android device

---

## Features

* Email Text Field
*  Password Text Field
*  Show/Hide Password
*  Login Button
*  Loading Animation
*  Login Success Snackbar
*  Invalid Login Snackbar
* Clean and Responsive UI
